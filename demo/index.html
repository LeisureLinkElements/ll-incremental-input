An example of <ll-incremental-input>:

The ll-incremental-input control uses a debounce as you're typing to fire off an event every 500 milliseconds.
The event contains the text from the input field in the detail.value property of the event. Pressing enter after typing will fire the event immediately.
One note, the event will only be fired if the text has changed. Pressing enter after the 500ms debounce will not fire the event again.
